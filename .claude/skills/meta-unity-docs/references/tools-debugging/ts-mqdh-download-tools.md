The Meta Quest Developer Hub (MQDH) is a one-stop shop for downloading and managing your Meta XR SDKs and tools.
Downloading SDKs, tools, and code samples directly from MQDH is the best way to discover new releases and stay up-to-date.
This guide explains how to download Meta XR SDK and tool packages and manage update notifications.

## Adding and updating packages

1. Under the **Downloads** tab, you should see a list of available SDKs and tools.

1. Optionally, you can filter the list by performing one of the following actions:
   - Select a platform in the **Platform** menu.
   - Search by name by entering text in the **Search downloads** text field.
   - Click the filter icon and then select the type of download.

1. To download the SDK or tool, click the **Download** button or similar next to the item.

    {:width="650px"}

   After the package is installed, it should be listed under **Installed**.

   {:width="650px"}

## Locate a downloaded package

When MQDH downloads packages, it saves them to the following directories by default:

- Windows path: `C:\Users\janedoe\AppData\Roaming\odh\packages`
- macOS path: `/Users/janedoe/Library/Application Support/odh/packages`

MQDH regularly scans this folder to check for updates, and notifies you to download the latest version.

## Configure notifications

Notifications help you stay connected with the Meta Quest development ecosystem.
MQDH sends you notifications based on interests and subscriptions that you signed up for on the [Meta Horizon Dashboard](/manage/settings/) settings page.

To manage your notifications, click **Settings** in the left sidebar, then select the **Notifications** tab. Set the types on or off, based on your preferences.

To mute notifications, click **Notifications** in the left sidebar, then click the bell icon to toggle them on or off.
