The MQDH File Manager centralizes file management in MQDH, including on-device apps, recorded metrics, images, and videos associated with the device and MQDH.

## Browsing Files

To open the File Manager, select **File Manager** from the left navigation pane in MQDH.

When a Meta Quest device is plugged in, the File Manager will show files related to that specific device.

While browsing files, a user can take several actions on a file. The actions are found in the context menu that becomes available when hovering over the file. Actions include the following:

* Open a file (Also happens when you double click on a file)
* Open a file in its containing folder with the user's native file manager
* Delete a file

## Local vs Device Files

Sections of the File Manager include:

* On Device - The parent folder of all device files
  * Videos - Contains videos captured and stored on the device
  * Images - Contains images captured and stored on the device
  * Recorded Metrics - Contains CSV files that are produced by the device when the Metrics Recording feature is turned on. Opening these files opens a browser window.
  * Apps - Contains APKs installed on the device from Unknown Sources (sideloaded apps not from the Meta Store)
* MQDH Local - The parent folder of all files local to MQDH
  * Videos - Contains videos captured on the device via MQDH, which are stored locally
  * Images - Contains images captured on the device via MQDH, which are stored locally
  * Perfetto - Contains Perfetto traces captured on the device via MQDH, which are stored locally
