## Overview

This guide describes guidelines and resources for creating a compelling VR audio experience in Unity using Wwise as the audio engine. It will walk you from setting up a very simple Wwise project using the Meta XR Audio SDK for Wwise to integrating that Wwise project into your game.

This guide focuses on the use of the Meta XR Audio SDK for Wwise but doesn't attempt to be a Wwise tutorial.

Audiokinetic provides tutorials and documentation on [their website](https://www.audiokinetic.com/en/learning) if you are new to Wwise.
[The Wwise-101 video series](https://www.audiokinetic.com/en/learning/videos/) is a great starting point for a beginner level that covers the higher level concepts and fundamental features.
 There's also a whole [section of the documentation](https://www.audiokinetic.com/en/library/edge/?source=Unity&id=index.html) that covers integrating Wwise into a Unity project.

By the end of this document, you’ll be able to:

- Download and install the Meta XR Audio SDK for Wwise.
- Get spatialized audio and room acoustics from the Sink Plugin.
- Adjust spatialization parameters for each sound source.
- Adjust where the room acoustics originate from, size of the room, room materials, and clutter settings.
- Play ambisonic audio.
