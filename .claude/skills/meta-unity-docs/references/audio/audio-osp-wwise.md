The Oculus Spatializer Plugin (OSP) is an add-on plugin for the Audiokinetic Wwise tool set that allows monophonic sound sources to be spatialized in 3D relative to the user's head location.

* **[Set Up the Oculus Spatializer for Wwise](/documentation/unity/audio-osp-wwise-overview/)**<br/>
Describes how to install, and update the OSP in the Wwise application.

* **[How to Use the Oculus Spatializer for Wwise](/documentation/unity/audio-osp-wwise-usage/)**<br/>
Describes how to use the OSP in the Wwise application.

* **[Integrating the Oculus Spatializer](/documentation/unity/audio-osp-wwise-integration/)**<br/>
Describes how to integrate Wwise libraries and plugin registration in your application code base.

* **[Configure Target Platform Plugins in Wwise Unity Projects](/documentation/unity/audio-osp-wwise-unity-install/)**<br/>
Describes how to add Oculus Spatializer plugin after installing the Wwise Integration Package to your Unity projects. 

For an overview of all of the audio tools offered by Meta, see    [Working with Audio in Unity](/documentation/unity/unity-audio/).  

For general background information on audio spatialization, see our [Introduction to Virtual Reality Audio](/resources/audio-intro-spatialization/).
