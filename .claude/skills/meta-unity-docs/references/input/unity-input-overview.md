On this page, we provide an overview of the SDKs that enable you to access and handle user input in a Horizon OS app built in Unity.

Apps developed on Meta XR SDKs can access and handle input from a user's head, hands, face, and voice using Meta Quest headset and Touch controller technology.

## Core SDK

Meta XR Core SDK includes all of the fundamental tools and assets needed to start building XR apps for Meta Quest headsets, including the [Project Setup Tool](/documentation/unity/unity-upst-overview), the [Meta XR Rig](/documentation/unity/unity-ovrcamerarig), and a number of low-level input features exposed by Core SDK APIs, like integrating [virtual keyboards](/documentation/unity/VK-unity-overview), and [accessing and handling user data directly from input sources](/documentation/unity/unity-ovrinput).

For more information about Meta XR Core SDK, see [Meta XR Core SDK download page](/downloads/package/meta-xr-core-sdk). For a detailed API reference for Meta XR Core SDK, see [Unity Core SDK Reference](/reference/unity/latest).

## Interaction SDK

Meta XR Interaction SDK includes prefabricated, customizable interaction components that you can leverage when developing interactive experiences for your end users.

For more information about Meta XR Interaction SDK, see [Interaction SDK Overview](/documentation/unity/unity-isdk-interaction-sdk-overview).

## Movement SDK

Meta XR Movement SDK enables you to incorporate body and face tracking into your app.

For more information about Meta XR Movement SDK, see [Movement SDK Overview](/documentation/unity/move-overview/).

## Voice SDK

Meta XR Voice SDK enables you to bring voice interactions to your app experiences.

For more information about Meta XR Voice SDK, see [Voice SDK Overview](/documentation/unity/voice-sdk-overview/).

## Haptics SDK

Meta XR Haptics SDK enables you to incorporate haptic feedback into applications in order to create immersive and engaging experiences.

For more information about Meta XR Haptics SDK, see [Haptics SDK Overview](/documentation/unity/unity-haptics-sdk).
