# Voice Sdk Traits

**Documentation Index:** Learn about voice sdk traits in this documentation.

---

---
title: "Built-In Traits"
description: "Reference the pre-trained trait types available in Voice SDK built-in natural language processing."
---

Currently, built-in traits are only available in English.

| Trait | Meaning |
|-|-|
| `wit$bye` | A binary trait that captures goodbye intents, such as “bye”. |
| `wit$greetings` | A binary trait that captures greeting intents, such as “bye” or "hello." |
| `wit$sentiment` | A trait that captures the sentiment in an utterance and returns positive, neutral or negative. For example, “Have an awesome day!” |
| `wit$thanks` | A binary trait that captures thankful intents, such as “thank you.”  |
