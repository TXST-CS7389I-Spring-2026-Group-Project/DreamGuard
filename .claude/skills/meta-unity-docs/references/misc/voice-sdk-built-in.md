Built-in Natural Language Processing (NLP) provides pre-trained intents, entities, and traits that are commonly used by developers. By utilizing these built-ins, you can quickly bootstrap your app during development. You can also include built-ins in your own custom Wit.ai app.

## Supported Languages for Built-In NLP

Voice SDK supports English (en) built-in NLP and offers an early preview of built-ins in the following languages that Meta Quest supports.
