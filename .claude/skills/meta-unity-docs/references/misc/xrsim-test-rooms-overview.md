## Overview

When developing MR apps, ensuring compatibility with various room setups is critical. This process can be challenging and time-consuming. XR Simulator simplifies this process. It provides nine rooms for testing purposes, including three hero rooms and six feature rooms, along with convenient tools for creating custom test environments.

Visit the [built-in rooms](/documentation/unity/xrsim-heroscenes/) page for details, and explore the [Synthetic Environment Builder](/documentation/unity/xrsim-synthetic-environment-builder/) to learn about constructing your own test rooms.
