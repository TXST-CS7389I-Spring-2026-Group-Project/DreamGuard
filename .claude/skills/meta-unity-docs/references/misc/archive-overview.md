The topics in this section contain reference information for retired (deprecated) features. These features are no longer supported outside of existing legacy apps in the Meta Horizon Store. The content here is retained for reference purposes for those legacy apps built using Unity.

Archived content will not be updated in the future. The topics in this section will be retained for a period of time. If you're using retired (deprecated) features, we recommend using supported alternatives.
