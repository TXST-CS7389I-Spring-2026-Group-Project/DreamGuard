## Overview

When you encounter issues in Meta XR Simulator, logs can provide crucial information for diagnosing and resolving the problem.

## Log Locations (Windows)

| Log Type | Location |
| --- | --- |
| Event Log  | %AppData%\MetaXR\MetaXrSimulator\logs |
| Crash Dump | %LocalAppData%\Oculus\MetaXRSimulator\crashes |

## Log Locations (macOS)

| Log Type | Location |
| --- | --- |
| Event Log  | ~/Library/Application Support/MetaXR/MetaXrSimulator/logs |
| Crash Dump | ~/Library/Logs/DiagnosticReports/Oculus/MetaXRSimulator |

## Logs Window

Logs are also available in the Logs window within Meta XR Simulator.
