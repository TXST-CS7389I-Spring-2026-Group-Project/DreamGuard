The Multiplayer Test Cases below cover Multiplayer scenarios that should all be supported for all apps in Meta Quest that support multiplayer features. While not a requirement at this time, this is the level of comfortable gameplay that Meta holds for any multiplayer game.

This grid describes the Join Intent Payload (across) and the App State (down).

Before implementing, review the [Group Presence Overview](/documentation/unity/ps-group-presence-overview/), [Destinations Overview](/documentation/unity/ps-destinations-overview/), [Travel Testing and Use Cases](/documentation/unity/ps-travel-testing-and-use-cases/), and [Multiplayer Testing Tool](/documentation/unity/ps-multiplayer-testing-tool/).
