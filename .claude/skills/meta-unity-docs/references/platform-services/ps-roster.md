## Roster

The roster helps users manage which friends they are playing with in-game. Your game must keep all user group presence data up to date to accurately reflect the current list of users in the roster.

When the game sets the current user's presence, users with the same lobby session ID are populated. The Meta Horizon platform then shows that list of users with shared group presence in the roster. The recommended best practice is to have users with the same lobby session ID if they are able to interact with each other in some way within the application. For example:
* They can talk to each other
* They can see each other

The Roster, and the users who are currently in the same session, can be viewed by bringing up the Meta Quest menu. The **Playing with you** window will display the current users and their presence information.
