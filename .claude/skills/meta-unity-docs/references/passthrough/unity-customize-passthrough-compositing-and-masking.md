By default, a passthrough layer uses automatic environment reconstruction, which covers the entire screen. There are multiple ways to combine passthrough and VR in interesting ways to create mixed reality experiences:

- Occlude virtual objects by real objects, and let passthrough show only in selected regions: [Occlusion](/documentation/unity/unity-customize-passthrough-passthrough-occlusions/).
- Create windows through which passthrough is visible: [Passthrough Windows](/documentation/unity/unity-customize-passthrough-passthrough-windows/).

These techniques are not mutually exclusive. They can be combined to create more sophisticated visualizations.
