This section contains pre-v62 tutorials for Interaction SDK that explain how to manually add interactions to your project. The [Quick Actions tutorial](/documentation/unity/unity-isdk-quick-actions/) is the recommended way to add interactions if you have v62 or higher of Interaction SDK, so we strongly recommend upgrading. However, for users who don't want to upgrade, it's still possible to manually add interactions, though the process is complicated and time-consuming.
