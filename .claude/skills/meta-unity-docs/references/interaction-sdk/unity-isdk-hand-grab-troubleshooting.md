## Overview

This topic describes some common issues and solutions when implementing [Hand Grab](/documentation/unity/unity-isdk-hand-grab-interaction/) and [Controller Grab](/documentation/unity/unity-isdk-grab-interaction/) interactions in your project.

## Why can't I edit fingers for a Hand Grab Intractable?

Make sure Gizmos are toggled on and visible. Although you can see the hand of the HandGrabIntractable, you can't see or edit the finger elements unless Gizmos are toggled on.
