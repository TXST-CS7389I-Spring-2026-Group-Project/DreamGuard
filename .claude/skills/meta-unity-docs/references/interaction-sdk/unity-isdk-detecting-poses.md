# Unity Isdk Detecting Poses

**Documentation Index:** Learn about unity isdk detecting poses in this documentation.

---

---
title: "Detecting Poses"
description: "Creating and detecting specific hand and body poses made by the player."
last_updated: "2025-08-07"
---

Interaction SDK provides the ability to detect whether the user is performing a specific pose. This is useful for triggering events, such as starting a game, or for enabling or disabling features. The topics below cover how to create and detect specific hand and body poses.
