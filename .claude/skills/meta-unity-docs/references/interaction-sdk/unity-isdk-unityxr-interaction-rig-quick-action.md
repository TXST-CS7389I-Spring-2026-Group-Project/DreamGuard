Interaction SDK provides the UnityXR Interaction Rig *Quick Action* utility, available via the right-click menu in the **Heirarchy** panel, to automate adding the UnityXR Comprehensive Interaction Rig to your scene.

This simplifies the process of adding a fully-functional interaction rig, making it easier for developers to create immersive experiences. In this guide, you'll learn how to use the quick action utility to add the UnityXR Comprehensive Interaction Rig.

## How does the UnityXR Interaction Rig Quick Action work?

The **UnityXR Comprehensive Rig Wizard**, which contains settings and required component options for configuring the UnityXR Interaction Rig Quick Action, is displayed when the UnityXR Interaction Rig Quick Action is selected. Once configured, the user can click **Create** and the quick action will handle creating all necessary components and *wiring* everything up to add the UnityXR Comprehensive Interaction Rig. The required components are explained below.

### Required Components

The comprehensive interaction rig requires the following components to be present in the scene:

- a **UnityXRCameraRig** component to add the comprehensive interaction rig components to.
