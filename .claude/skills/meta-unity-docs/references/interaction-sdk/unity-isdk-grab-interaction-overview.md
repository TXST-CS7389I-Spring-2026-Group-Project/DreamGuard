# Unity Isdk Grab Interaction Overview

**Documentation Index:** Implement grab interactions in Unity using hands, controllers, or controller-driven hands for immersive experiences.

---

---
title: "Grab Interactions"
description: "Family of interactions that enable you to grab and release objects within reach of the player."
last_updated: "2025-08-07"
---

Objects within reach of the player can be grabbed using hands, controllers, or controller-driven hands, enabling an immersive experience regardless of the input device used.
