## Avatars SDK and Interaction SDK sample project

The [Avatars and Interaction SDK sample project](https://github.com/oculus-samples/Unity-MetaXRInteractionSDK-AvatarSample) on GitHub contains scenes
that show how to use interactions while the player avatar's movements are driven by the Avatar SDK.
