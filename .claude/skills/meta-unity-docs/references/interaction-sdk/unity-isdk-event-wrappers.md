# Unity Isdk Event Wrappers

**Documentation Index:** Learn about unity isdk event wrappers in this documentation.

---

---
title: "Event Wrappers"
description: "React to active state, interactor, interactable, and canvas events by wiring Unity event callbacks to Interaction SDK components."
---

You can attach event wrappers to Interaction SDK components like active state, interactors, and canvases to trigger custom logic when specific events occur. To learn more about the events that occur during the interaction lifecycle, see [Interactor Interactable Lifecycle](/documentation/unity/unity-isdk-interactor-interactable-lifecycle/).
