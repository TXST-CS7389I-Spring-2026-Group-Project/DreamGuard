## Overview

This topic describes some common issues and solutions when implementing Interaction SDK in your project.

## Why can't I move my thumb or index finger by touching the trigger or buttons?

When using **OpenXR** and **ControllersAsHands**, it is not possible to move the thumb or index finger by touching the trigger or buttons. A press is required. This is a documented issue with the **OpenXR** library.
