The Meta Avatars SDK has reached End-of-Feature (EOF) status. **[Version 40.0.1](/downloads/package/meta-avatars-sdk/) is the final release.** While no further SDK versions, new features, or API additions are planned, all existing integrations continue to work and backend services remain fully operational. Use the Meta Avatars SDK to create expressive, diverse, and customizable avatar identities for the Meta ecosystem, Unity VR apps, and other multiplayer experiences.

With the Meta Avatars SDK you can:

- Select from packaged avatar presets for quick iteration, or enable user-created avatars for more customization options.
- Use advanced body tracking capabilities to create realistic avatar features and poses.
- Create more natural facial expressions by using Meta Quest Pro eye tracking and face tracking.
- Customize your Avatar’s look and feel with extensive clothing and accessory options from the Avatars Store.

Read the [best practices](/documentation/unity/meta-avatars-best-practices/) guide, then download the Meta Avatars SDK below.

## Downloads

You can download the Meta Avatars SDK in two ways:
- The [Meta Avatars SDK](/downloads/package/meta-avatars-sdk/) is available as a separate package for Unity on the Meta Downloads page.
- You can also [download the Meta Avatars SDK package from Meta Quest Developer Hub (MQDH)](/documentation/unity/ts-mqdh-download-tools/). This way, the MQDH Package Manager can notify you whenever there is a new update available.

For more information about integration capabilities, read our guide for [configuring Unity projects to work with the Meta Avatars SDK](/documentation/unity/meta-avatars-config-project/), or learn how to [integrate Avatars into any networking system](/documentation/unity/meta-avatars-networking/).
