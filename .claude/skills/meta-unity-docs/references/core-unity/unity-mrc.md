Mixed reality capture and virtual camera support for Unity apps now uses LIV.tv's tooling and SDKs. You can capture authentic content using a PC with an external camera in both immersive and MR apps.

For more information, see:
* [Capture High-Quality Mixed Reality Footage with LIV SDK v2.0](/blog/meta-quest-developer-LIV-mixed-reality-capture-support-sdk/) in the Meta Horizon developer blog
* [LIV SDK for Mixed Reality Applications](https://mrc-docs.liv.tv/intro/liv-sdk)

See [MRC best practices for development and video production](/resources/mixed-reality-technical-and-trailer-video) for tips on solving technical challenges related to mixed reality capture.

## If you are currently using Meta's older MRC libraries

If you already use our older MRC tooling, the transition to LIV.tv will not impact any existing MRC integrations. To help drive better capture and virtual camera solutions throughout the Meta Horizon ecosystem, we are removing the MRC download links from our documentation and recommend integrating LIV.tv’s tooling to support your future MRC needs.
