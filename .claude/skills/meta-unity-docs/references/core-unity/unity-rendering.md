This guide describes advanced rendering features that improve performance and build a more immersive, realistic experience.

- **[Fixed Foveated Rendering](/documentation/unity/unity-fixed-foveated-rendering)** - Create a realistic immersive experience and improve performance by using Fixed Foveated Rendering to render lower quality at the peripherals of headset displays.
- **[Display Frequencies](/documentation/unity/unity-set-disp-freq)** - Improve the quality of visuals using high frequencies that prevent judder and flickering black areas.
- **[Enable Phase Sync](/documentation/unity/enable-phase-sync)** - Reduce latency by using this frame timing management technique.
- **[VR Compositor Layers](/documentation/unity/unity-ovroverlay)** - Render some types of content such as text, video, and textures with high quality textures by sampling from a layer's texture directly rather than using eye buffers.
- **[Single Pass Stereo Rendering and Stereo Instancing](/documentation/unity/unity-single-pass)** - Optimize CPU usage by automatically mirroring the eye buffer and make appropriate modifications for a binocular viewing experience.
- **[Set Specific Color Space](/documentation/unity/unity-color-space)** - Create a realistic visual experience by defining color capabilities for a headset using the Oculus color space.
