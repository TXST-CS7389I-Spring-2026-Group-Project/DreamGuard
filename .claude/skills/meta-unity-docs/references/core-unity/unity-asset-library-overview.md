</oc-devui-note>

The Asset Library is a collection of 3D models generated by AI that you can use to rapidly prototype
and produce your scenes in Unity Editor. All assets are free to use and provide four levels of detail (LOD) to help you
optimize your scene for performance or detail.

This guide provides instructions on how to search for assets and add them to your project.

## Prerequisites

Before proceeding with this tutorial, complete the setup steps outlined in [Set up Unity for VR development](/documentation/unity/unity-project-setup)
to create a project with the necessary dependencies.

Confirm that Meta XR Core SDK v83 or later is installed in your project.

## Using the Asset Library

### Step 1: Access the library

To open the library’s search interface, select **Meta XR Tools** > **Asset Library** from the menu at the top of your editor.

{:width="600px"}

### Step 2: Log into your Meta account

After you select **Asset Library**, you'll be prompted to log into your Meta account.
Select **Log into existing Meta account** and submit your credentials or register for a new account.

### Step 3: Search the library for assets

Enter a description of the asset you want and click **Search**. The library displays a list of assets
matching your search text.

{:width="590px"}

### Step 4: Select an asset and add it to your scene

1. Select the Level of Detail (LOD) setting that best meets your visual fidelity and performance balance requirements.

   For example, **LOD 0** is the most detailed and has the highest polycount, which increases CPU and memory usage.
   **LOD 3** is the least detailed and uses the least CPU and memory.

   {:width="600px"}

2. Click **+** (plus symbol) in the bottom right corner to add the asset to the current scene.
   Confirm that you see the asset in the **Hierarchy** pane.

   {:width="600px"}

## Troubleshooting

### Unable to locate the Asset Library

If the Asset Library is missing from the menu, make sure that the Meta XR Core SDK v83 or later is installed
in your project. Restart Unity Editor and open your project to refresh the menu options.

### Search returned a system error

If your search returns with an error, click **Try again** to continue. This resubmits the search, which might avoid
transient issues.
