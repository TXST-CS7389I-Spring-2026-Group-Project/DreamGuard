This document provides an overview of Meta's sample Unity XR projects.

Reality Labs' engineering teams have developed various sample projects to assist in creating XR apps with Unity. These projects differ in complexity and content, catering to developers with varying levels of expertise, at different development stages, and with different feature needs. We have categorized these projects to help you easily find the sample you need.

## UPM Package Samples

The following Meta XR SDKs include sample projects to help you get started with specific SDK features:

- [Meta XR Core SDK](/downloads/package/meta-xr-core-sdk)
- [Meta XR Interaction SDK](/downloads/package/meta-xr-interaction-sdk-ovr-integration/)
- [Meta XR Platform SDK](/downloads/package/meta-xr-platform-sdk/)
- [Meta XR Audio SDK](/downloads/package/meta-xr-audio-sdk)
- [Meta XR Mixed Reality Utility Kit](/downloads/package/meta-xr-mr-utility-kit-upm/)
- [Meta XR Haptics SDK](/downloads/package/meta-haptics-sdk-unity/)
- [Meta XR Voice SDK](/downloads/package/meta-voice-sdk/)

You can import sample projects directly from these UPM packages within the Unity Editor.

For more details on UPM package samples, including instructions on importing samples into a project, see [UPM Package Samples Overview](/documentation/unity/unity-package-samples/).

## Starter Samples

The Starter Samples project offers sample scenes to help you get started with core XR features.

These samples are available on GitHub in the [Unity-StarterSamples repo](https://github.com/oculus-samples/Unity-StarterSamples).

For more details on the Starter Samples project, see [Starter Samples Overview](/documentation/unity/unity-starter-samples/).

## Motifs

Motifs are sample projects that demonstrate specific feature implementations and best practices.

They are available on GitHub in the [Unity-MRMotifs repo](https://github.com/oculus-samples/Unity-MRMotifs).

For more details on Motifs, see [Mixed Reality Motifs Overview](/documentation/unity/unity-mrmotifs-overview/).

## Showcases

Showcases are complex sample projects that demonstrate multiple features integrated into [Store](https://www.meta.com/experiences/)-ready apps.

For an updated list of all GitHub-based projects, including showcases, see [Code samples for Unity](/code-samples/unity).

## Other Samples

For a complete list of samples in the [oculus-samples](https://github.com/oculus-samples) GitHub organization, see [Code samples for Unity](/code-samples/unity).
