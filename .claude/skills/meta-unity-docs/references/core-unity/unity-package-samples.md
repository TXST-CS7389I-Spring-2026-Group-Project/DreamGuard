## Import samples from UPM packages

To import samples for a UPM package, following these instructions:

1. Navigate to **Window** > **Package Management** > **Package Manager**.
2. At the top left of the window, make sure that the package source filter is set to **Packages: In Project**.
3. Select an installed package to show package details. Installed packages are marked with a check.
4. If the installed package has samples, a **Samples** section will appear either under the package description or as a section tab, depending on the Unity Editor version.
5. Go to the **Samples** tab, and select **Import** next to each sample project that you want to import into your editor.
6. After importing is complete, you can find all imported sample projects located in your project under **Assets** > **Samples** > **Meta XR __ SDK**.

## Example: Meta XR Core SDK Sample Scenes project

Suppose that you have installed the Meta XR Core SDK UPM package, and you want to import the Sample Scenes project that is shipped with the package:

1. In the **Package Manager**, select the **Meta XR Core SDK** package from the list of packages in your project.

    {:width="500px"}

2. On the package's page in the **Package Manager**, select the **Samples** tab, and then select **Import** next to the **Sample Scenes** project.

    {:width="500px"}

3. Close the **Package Manager** and look in your project under **Assets** > **Samples** > **Meta XR Core SDK**.

    {:width="500px"}

4. Double-click any scene to open it in the Unity Editor.
